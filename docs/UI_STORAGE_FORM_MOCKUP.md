## Key Visual Features

### 1. **Dropdown Items** (Rich Display)
- Icon on the left (HardDrive/Cloud/Database)
- Title in bold
- Description in smaller, muted text
- Proper spacing and alignment

### 2. **Dynamic Fields**
- Fields appear/disappear based on storage type
- Smooth transition (no page jump)
- Clear visual grouping for cloud-specific fields

### 3. **Field Labels**
- Context-aware (changes based on storage type)
- "Base Directory" â†’ "S3 Bucket URI" â†’ "GCS Bucket URI"
- Clear hierarchy with font weights

### 4. **Input Types**
- Regular Input for short text (IDs, names, regions)
- Textarea for large text (JSON keys, secrets)
- Monospace font for technical content

### 5. **Help Text**
- Below every field
- Muted color to not overwhelm
- Actionable examples in placeholders

### 6. **Validation**
- Required fields marked with *
- Pattern validation on Storage ID
- Disabled fields when editing
- Clear error messages

### 7. **Responsive Design**
- Max-width to prevent too-wide dialogs
- Proper spacing between fields
- Scrollable content if needed
